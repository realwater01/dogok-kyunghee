# 의료광고 심의 대응 문구 가이드

도곡경희한의원 홈페이지의 모든 산문(카피·설명·FAQ·메타데이터·llms.txt)은
**의료법 제56조(의료광고 금지)** 와 한의사 의료광고 사전심의 기준을 따른다.
새 문구를 만들거나 기존 문구를 고칠 때 아래 기준을 반드시 적용한다.

## 1. 쓰지 말 것 (심의에서 걸리는 표현)

- **최상급·배타성**: 최고, 최상, 최첨단, 1등, 유일, 제일, 가장, 베스트, No.1, 명의, 권위자, 상위 N%
- **효과 보장·단정**: 완치, 100%, 확실, 보장, 영구적, 반드시, 즉시
- **근본 치료 단정**: "근본적으로 치료합니다", "근본부터 치료합니다", "근본 원인을 치료합니다", "극대화"
- **부작용 없음·안전 단정**: 부작용 없는, 무통, 안전한 치료, 안전하게 치료
- **비교·비방**: 다른 한의원/병원보다 우수, 타 의료기관과 달리
- **환자 후기·치료경험담**: "○○ 환자가 나았어요" 류, 전후 사진(맥락 없는)
- **검증되지 않은 신의료기술**, 법적 근거 없는 자격·인증 표시

## 2. 권장 표현 (안전한 톤)

- 효과는 **돕는·완화 톤**으로: "~을 돕습니다", "~할 수 있습니다", "~에 중점을 둡니다", "회복을 돕습니다"
- 원인 관련: "근본적으로 치료" 대신 **"원인에 접근합니다", "원인 개선에 중점을 둡니다", "균형 회복을 돕습니다"**
- 컨디션/효과: "최상" 대신 **"좋은 컨디션"**, "극대화" 대신 **"도움이 되도록"**
- 자격·경력·논문은 **사실만** 표기 (경희대 학·석·박사, 한방내과 전문의, 임상 경력, 발표 논문 등)
- 부작용 등 **중요 정보는 누락하지 않는다** (누락도 위반)

## 3. 치환 예시 (이번 점검에서 실제 적용)

| 금지 → | 안전 |
|---|---|
| 한약 치료 상위 1% 자부심 | 한약, 정성을 다해 짓습니다 |
| 최상의 자연치료법입니다 | 자연에 가까운 치료법입니다 |
| 가장 평온한 공간 | 편안한 공간 |
| 근본적으로 치료합니다 | 개선을 돕습니다 / 원인에 접근합니다 |
| 근본부터 치료합니다 | 균형 회복에 중점을 둡니다 |
| 부작용 걱정 없이 안전하게 치료 | 아이 상태에 맞춰 신중하게 치료 |
| 안전하고 효과적인 한약치료 | 개인에 맞춘 한약 치료 |
| 치료 효과를 극대화합니다 | 치료에 도움이 되도록 합니다 |
| 최상의 컨디션 | 좋은 컨디션 |
| 가장 효과적이지만 | 도움이 되지만 |

## 4. 적용 범위

화면에 보이는 카피뿐 아니라 **SEO 메타데이터(title/description), 구조화데이터(site.ts),
llms.txt** 까지 동일 기준으로 맞춘다. 화면과 메타데이터가 어긋나지 않게 함께 수정한다.

## 5. 면책 안내 (권장)

치료 효과는 개인차가 있음을 알리는 문구를 적절한 위치에 두면 오인 소지를 줄일 수 있다.
(예: "치료 효과는 개인의 상태에 따라 다를 수 있습니다.")

> 본 가이드는 일반적 심의 실무 기준 정리이며, 최종 표현은 대한한의사협회 의료광고 사전심의로 확정한다.
